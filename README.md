# Collab Draw

Collab Draw is a simple real-time collaborative drawing application where multiple users can draw together on a shared canvas.

## About
Collab Draw allows users to draw together in real time. It’s built to be lightweight, fast, and easy to use for brainstorming, sketching, or collaborative note-taking.

## Tech Stack
- React / Next.js (frontend)
- Node.js (backend)
- WebSockets (real-time communication)
- TypeScript
- Tailwind CSS (styling)

## Features
- Real-time collaborative drawing
- Color and brush size options
- Eraser and clear canvas tools
- Room-based collaboration
- Responsive interface

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/AniruddhaM18/collab-draw.git
cd collab-draw

Install dependencies
```bash
pnpm install

Run the development server
pnpm run dev


The app will start locally, usually at http://localhost:3000
.

Folder Structure
collab-draw/

```
